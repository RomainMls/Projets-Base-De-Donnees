# Projet INFO0009 - Base de données

Ce dépôt regroupe nos deux travaux du cours INFO0009-2 Base de données. Le sujet tourne autour d'un système ferroviaire : d'abord le modéliser proprement, puis l'implémenter avec une base MySQL et une petite interface web en PHP.

L'idée est surtout de montrer le cheminement entre le modèle, les contraintes SQL, les données et les pages qui manipulent la BDD.

## Contenu

- `partie1_modelisation/` : rapport de modélisation du système ferroviaire.
- `partie2_implementation/` : implémentation MySQL/PHP avec Docker, scripts SQL, données CSV et interface web.
