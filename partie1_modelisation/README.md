# Partie 1 - Modélisation

Cette première partie correspond au travail de modélisation du système ferroviaire. Le but était de partir de l'énoncé du cours et de construire un modèle de données cohérent avant de passer à l'implémentation.

Dans le dossier, on retrouve surtout :

- `enonce.pdf` : le problème posé
- `report.pdf` : notre rapport avec les choix de modélisation.
- `feedback.pdf` : le retour reçu sur cette partie.

Ce n'est pas la partie qui se lance, mais c'est celle qui explique les bases du projet : les entités, les relations, les contraintes et les décisions prises avant d'écrire le SQL.
